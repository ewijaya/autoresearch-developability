# autoresearch-developability

Autoresearch for peptide multi-objective developability optimization.

This repository is a public-data proof-of-concept inspired explicitly by Andrej Karpathy's autoresearch:
https://github.com/karpathy/autoresearch

It is also grounded in lessons from:
https://github.com/ewijaya/autoresearch-mol

See `docs/PRD.md` for the kick-start product requirements document.
