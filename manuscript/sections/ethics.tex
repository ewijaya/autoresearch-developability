\section{Ethics Statement}
\label{sec:ethics}

This work uses exclusively public datasets (DBAASP, ToxinPred3, HLP) and involves no human subjects, animal experiments, or personal data.
The system is a ranking tool for candidate prioritization, not a generative model; it does not design novel molecules and therefore poses no dual-use biosecurity risk.
The Codex agent was used through the standard OpenAI API with no adversarial prompting.
All claims are scoped to the public benchmark; we do not claim clinical relevance or therapeutic implications.
